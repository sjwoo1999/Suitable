# Suitable (슈터블)

**당신의 시작을 슈터블하게.**
Suitable은 대학생과 사회초년생을 위한 프리미엄 정장 렌탈 서비스입니다. 중요한 순간, 부담스러운 구매 대신 합리적인 렌탈로 완벽한 스타일을 완성하세요.

## 🌟 주요 기능 (Key Features)

*   **슈터블 픽업 (Campus Pickup)**: 학교 정문 픽업 스테이션에서 공강 시간에 간편하게 수령할 수 있습니다.
*   **슈터블 AI 핏 (AI Size Fit)**: 사진 한 장으로 내 몸에 딱 맞는 최적의 사이즈를 추천해 드립니다.
*   **슈터블 케어 (No Laundry)**: 세탁과 관리 걱정 없이 입은 그대로 반납하면 됩니다.
*   **ESG 가치 실현**: 정장 공유를 통해 자원 낭비를 줄이고 지속 가능한 소비를 지향합니다.

## 🎨 디자인 시스템 (Design System)

Suitable은 신뢰감을 주는 **Navy**와 산뜻한 시작을 의미하는 **Mint** 컬러를 메인으로 사용합니다.

*   **Colors**:
    *   Primary: Navy (`#1a2b4c`)
    *   Accent: Mint (`#4cd9b0`)
    *   Background: White (`#ffffff`), Gray (`#f5f5f5`)
*   **Typography**: Pretendard (프리텐다드)

## 💰 요금제 (Pricing)

*   **Individual & Friends**: 1회 체험권, 슈퍼 얼리버드, 커플/친구 패키지 등 다양한 개인 및 소그룹 요금제.
*   **Group Packages**: 학과, 동아리 등 10인 이상 단체를 위한 특별 할인 패키지.
*   **VIP Campus Pass**: 한 학기 동안 자유롭게 이용할 수 있는 VIP 전용 패스.

## 🛠️ 기술 스택 (Tech Stack)

*   **HTML5**: 시맨틱 마크업 구조.
*   **CSS3**: Vanilla CSS, Flexbox 레이아웃, 반응형 디자인 고려.
*   **Font**: Pretendard (CDN).

## 🚀 실행 방법 (How to Run)

이 프로젝트는 정적 웹사이트입니다. 별도의 빌드 과정 없이 바로 실행할 수 있습니다.

1.  프로젝트 폴더를 다운로드합니다.
2.  `index.html` 파일을 웹 브라우저(Chrome, Safari, Edge 등)에서 엽니다.

## 📂 프로젝트 구조 (Project Structure)

```
Suitable/
├── index.html          # 메인 랜딩 페이지
├── image_prompts.md    # 이미지 생성 프롬프트 모음
└── images/             # 웹사이트에 사용된 이미지 리소스 폴더
```
